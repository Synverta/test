# 斜线分割图片网页 - 使用说明

## 项目概述

这是一个使用HTML和CSS创建的网页，通过一条对角线将页面分为两个部分，每个部分显示一张图片，点击图片可以跳转到不同的网页。

## 文件说明

- **index.html** - 主网页文件，包含完整的HTML结构和CSS样式
- **img1.jpg** - 左上三角形区域的图片
- **img2.jpg** - 右下三角形区域的图片

## 技术实现

### 1. 斜线分割效果

使用CSS的 `clip-path` 属性实现斜线分割：

```css
/* 左侧区域：从左上角到右上角再到左下角的三角形 */
.left-section {
    clip-path: polygon(0 0, 100% 0, 0 100%);
}

/* 右侧区域：从右上角到右下角再到左下角的三角形 */
.right-section {
    clip-path: polygon(100% 0, 100% 100%, 0 100%);
}
```

两个三角形共享一条从右上到左下的对角线，形成完美的斜线分割效果。

### 2. 响应式布局

- 使用 `100vh` (viewport height) 确保网页占满整个屏幕
- 使用 `object-fit: cover` 确保图片按比例填充区域
- 支持移动设备和桌面浏览器

### 3. 交互效果

- **点击跳转**：每张图片都是可点击的链接，使用 `<a>` 标签包裹
- **新窗口打开**：使用 `target="_blank"` 在新标签页打开链接
- **悬停效果**：鼠标悬停时图片轻微放大（1.05倍），提升用户体验

## 使用方法

### 快速开始

1. 将以下文件放在同一目录：
   - index.html
   - img1.jpg（您的第一张图片）
   - img2.jpg（您的第二张图片）

2. 在浏览器中打开 `index.html` 文件

### 自定义配置

#### 更换图片

替换 `img1.jpg` 和 `img2.jpg` 为您自己的图片文件，或修改HTML中的图片路径：

```html
<img src="您的图片路径.jpg" alt="图片描述">
```

#### 修改链接地址

在 `index.html` 中找到 `<a>` 标签，修改 `href` 属性：

```html
<!-- 修改这里的网址 -->
<a href="https://您的网站.com" target="_blank" class="image-link">
```

#### 调整斜线角度

如果想要不同角度的分割线，可以修改 `clip-path` 的坐标值：

```css
/* 示例：更陡峭的斜线（从中间分割） */
.left-section {
    clip-path: polygon(0 0, 50% 0, 0 100%);
}
.right-section {
    clip-path: polygon(50% 0, 100% 0, 100% 100%, 0 100%);
}
```

#### 移除文字说明

如果不需要图片上的文字提示，删除以下代码：

```html
<div class="image-overlay">点击访问页面1</div>
```

## 常见问题

**Q: 图片显示不完整怎么办？**  
A: 使用 `object-fit: cover` 会自动裁剪图片以填充区域。建议使用高分辨率图片（推荐 1920x1080 或更大）。

**Q: 能否改为水平或垂直分割？**  
A: 可以，修改 `clip-path` 的多边形坐标即可实现不同的分割方式。

**Q: 如何在同一页面打开链接而不是新标签页？**  
A: 删除或修改 `target="_blank"` 属性。

**Q: 可以添加更多图片区域吗？**  
A: 可以，但需要更复杂的 `clip-path` 设计来创建多个不重叠的区域。
